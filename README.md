## Description

Pecha tool web app for speech to text

## Install

Requires to have node and npm installed.

`$ npm install`

## Development

To add dummy data, need to run seeder file (prisma/seed.js)

`npx prisma db seed`

 Browse your data

 `npx prisma studio`
